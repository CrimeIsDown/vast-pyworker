vast-pyworker utilizes the ai-dock: Stable Diffusion WebUI Docker Image ([README](https://cloud.vast.ai/template/readme/16109ada1a6101760c160543d3867d43)) and runs it using api-mode. You can install different models by changing the PROVISIONING_SCRIPT environment variable used. For more information on this see the [README](https://cloud.vast.ai/template/readme/16109ada1a6101760c160543d3867d43) of the template. The some parameters had to be changed from the default values in order to run the template in API mode.
